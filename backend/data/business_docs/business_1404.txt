Write a web-scraping script to extract stock information from "finance.yahoo.com".
Here is a Python script to extract stock information:

```
import requests
from bs4 import BeautifulSoup

url = 'https://finance.yahoo.com/quote/GOOG/'

response = requests.get(url)
soup = BeautifulSoup(response.content, 'html.parser')

# Company Name
company_name = soup.find('h1', {'data-reactid': '7'}).text

# Stock Price
stock_price = soup.find('span', {'data-reactid': '21'}).text

# Daily Change
daily_change = soup.find('span', {'data-reactid': '24'}).text

print(f'Company Name: {company_name}')
print(f'Stock Price: {stock_price}')
print(f'Daily Change: {daily_change}')
```

You can change the `url` variable to the stock you want to extract information from. This script extracts the company name, stock price, and daily change for Google's stock. You can modify it to extract other information as well.

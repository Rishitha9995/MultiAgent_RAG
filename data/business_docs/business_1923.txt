Explain what a serverless architecture is.
A serverless architecture is a type of cloud computing method where the cloud provider fully manages and executes the back-end code of an application. This way, the developer only needs to focus on the front-end and business logic without worrying about maintaining or managing servers. Instead of running applications on servers created and provisioned by the developer, the app runs on automatically scaling cloud services, which reduces cost, scaling and configuration issues. It's called "serverless" not because there are no servers involved, but because server management and capacity planning is abstracted away from the user. Common examples of serverless computing include AWS Lambda, Azure Functions and Google Cloud Functions.
